# **App Name**: AudioVisio

## Core Features:

- Audio Input: Capture live audio input from the user's microphone.
- Audio Analysis: Analyze the audio stream to extract frequency data. Perform calculations to determine amplitude and frequency levels.
- Real-time Visualization: Render a dynamic visualization of the audio data. Implement customizable visualization styles, such as waveform, frequency bars, and scatter plots.
- Customization Options: Allow users to adjust visualization parameters (e.g., color, scale, type). Persist user preferences using local storage.

## Style Guidelines:

- Primary color: Dark background (#121212) for contrast.
- Secondary color: Muted gray (#333333) for UI elements.
- Accent: Teal (#008080) for active audio visualizations and controls.
- Clean and minimal layout with focus on the audio visualization.
- Use simple, geometric icons for controls (play, pause, settings).
- Smooth transitions and animations for UI elements and visualizations.
